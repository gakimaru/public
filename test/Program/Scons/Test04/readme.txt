※まだまだ課題多し
  ・ソースファイルの自動ピックアップ ⇒ globオブジェクトを使っても、階層が深くなるとうまくいかない。
  ・ヘッダーファイルの依存関係自動検出 ⇒ .cpp ならうまくいくが、独自拡張子での扱いがうまくいかない。
  ・出力先フォルダの指定 ⇒ パスを加工する必要あり。現状方法不明。
  ・もう一つ独自プリプロセッサを通し（CRC値展開など）、最後にスクリプトをコンパイルする ⇒ 未対応
  ・MD5ハッシュによるファイルの更新確認 ⇒ 未確認
  ・強制インクルードファイルを依存ファイルとみなす ⇒ 対応していなさそう
